A simple one-click solution to quickly duplicate any WordPress Post, Page, or Custom Post Type, including Custom Fields.

== Description ==

Quick Clone is a lightweight and powerful WordPress plugin designed to boost your content creation productivity. 

Instead of manually recreating content, Quick Clone adds a simple "Duplicate" link in your post list table and post edit screen, making content cloning instant.

= Key Features =

* **One-Click Duplication:** Create a perfect copy of any post type instantly.
* **Clones Everything:** Copies post content, title, featured image, post status, and all associated **Post Meta (Custom Fields)**.
* **Taxonomy Support:** Correctly clones all Categories, Tags, and Custom Taxonomies.
* **Safe and Secure:** Follows all WordPress coding standards and uses Nonces for security.
* **Works with:** Posts, Pages, and all Custom Post Types.

== Installation ==

1.  Go to **Plugins -> Add New** in your WordPress Admin Dashboard.
2.  Search for **"Quick Clone"** (once it is live).
3.  Click **"Install Now"** and then **"Activate"**.
4.  You will now see a **"Duplicate"** link when you hover over any post or page in the list view.
